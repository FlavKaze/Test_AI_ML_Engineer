Initial Setup

This project analyzes operational data from industrial equipment contained in an Excel file. The monitored metrics include Temperature, Pressure, Vibration on three axes (X, Y, Z), and Frequency. The dataset also includes operational configuration information (Preset_1 and Preset_2) along with a binary failure indicator.

The data structure provides a comprehensive view of equipment behavior under different operating conditions, allowing for identification of failure patterns and predictive modeling.

1. Overall Failure Frequency

The equipment experienced a documented number of failure events throughout the analyzed period. This count serves as the baseline for understanding the proportion of anomalous events in the dataset. Knowing the failure rate is essential for assessing the balance of the dataset and understanding how often problematic behavior occurs in normal operations.

This baseline frequency helps in evaluating model performance and setting realistic expectations for failure prediction accuracy.

2. Failure Patterns by Operating Configuration

When data was grouped by operational configuration type, significant variations in failure rates emerged across different preset combinations. Certain configuration pairs demonstrated substantially higher failure counts than others.

Notable findings:
  • Configurations 1-5 and 3-5 exhibited the highest failure rates
  • Configuration 3-4 operated without any recorded failures
  • Equipment that failed consistently showed higher metric readings compared to normally operating equipment

This correlation suggests that elevated sensor readings across multiple parameters are strongly associated with equipment failure, providing an early warning opportunity.

3. Correlation and Effect Size Analysis

Correlation analysis using Pearson's method identified VibrationY as the strongest single indicator of failure. Equipment with elevated VibrationY readings showed strong correlation with failure events. While other metrics also demonstrated relationship with failures, their correlation was notably weaker.

Effect Size Analysis (Cohen's d): The statistical distributions of measured parameters differed substantially between failed and non-failed equipment, confirming that these metrics genuinely distinguish between operational states rather than showing random variation.

The rank order of metric importance by correlation strength provides guidance for focusing monitoring efforts and tuning alarm thresholds.

4. Predictive Modeling

Four different machine learning models were trained and evaluated: Logistic Regression, Random Forest, Support Vector Machine (SVM), and XGBoost. The dataset was split 75% for training and 25% for testing, with stratification applied to preserve class balance.

Model Performance Winner: Logistic Regression

Logistic Regression outperformed the other three models across key metrics including accuracy, recall, and ROC-AUC score. This is particularly significant because recall—the ability to identify failures that actually occur—is critical in failure prediction scenarios. Missing actual failures is more costly than occasional false alarms.

The success of a simpler model suggests that failure in this equipment follows relatively straightforward patterns that don't require complex non-linear modeling.

5. Variable Importance

By extracting coefficients from the Logistic Regression model, the relative importance of each input variable was quantified. Frequency emerged as the variable with the strongest influence on failure predictions, with higher frequency values substantially increasing the predicted probability of failure.

Importance Ranking:
  1. Frequency - strongest impact on predictions
  2. VibrationY - second strongest, aligning with correlation analysis
  3. Remaining metrics - lower but measurable influence

This ranking corroborates the correlation analysis findings and provides actionable guidance for which parameters deserve the most attention from operators.

Key Takeaways

The analysis reveals several important insights:

  • Mode-Dependent Risk: Specific operating modes are inherently higher-risk, with configurations 1-5 and 3-5 showing elevated failure rates
  • Early Warning Signals: Elevated values in Frequency and VibrationY function as reliable early indicators of impending failure
  • Effective Simplicity: A straightforward Logistic Regression model successfully captures the underlying failure patterns and identifies the majority of failures in advance
  • Actionable Metrics: Focus monitoring and maintenance efforts on Frequency and VibrationY for maximum impact
